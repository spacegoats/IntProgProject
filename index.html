REDDIT ANALYSIS BOT

  • HOME
  • STATS

An automated reddit bot that analyzes the front page submissions and to which subreddit they were posted to every 30 minutes. From there the bot uploads its findings to a cloud hosted database and tallies up every repeating subreddit. This is then pushed to a web front for organized viewing.

WORKFLOW

  • Heroku deployed python bot wakes and tallies top 25 submission subreddit attributes.
  • Inserts subreddit, a count value and a datestamp into a stats table in a database.
  • Utilizes PHP to 'post' these values to a webpage and sorts them simultaneously.

TOOLS USED

  • PRAW: reddit's python based API.
  • Heroku: badass cloud processing service with tons of add ons.
  • Postgres: cloud based database system, mySQL but with awesome heroku integration.
  • PHP: whatever,it's just PHP.
  • Bluehost: web hosting
